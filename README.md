# Exercice HTML : Site de Nutrition

Ce projet est un exercice pratique (TP) visant à créer un site web statique structuré à l'aide de tableaux HTML. Le site fournit des conseils nutritionnels sur la répartition des repas au cours de la journée.

## Description

L'objectif de cet exercice est de maîtriser la mise en page (layout) classique utilisant la balise `<table>`, une technique historique du web. Le site est optimisé pour une résolution de 800x600 pixels, avec une largeur fixe de 740 pixels.

Le site est composé de 4 pages principales accessibles via un menu de navigation latéral :
1.  **Accueil** : Introduction sur la fréquence des repas.
2.  **Petit déjeuner** : L'importance du premier repas et sa composition.
3.  **Déjeuner** : Conseils pour les repas du midi, au travail ou à l'extérieur.
4.  **Diner** : Idées de repas légers pour le soir.
